# Hyperfinity Fonts

Place the Roobert font files in this directory:

- `Roobert-Regular.woff2`
- `Roobert-Medium.woff2`
- `Roobert-SemiBold.woff2`
- `Roobert-Bold.woff2`

You can obtain the Roobert font from:
- https://displaay.net/typeface/roobert/

If the font files are not available, the app will fall back to system fonts.
